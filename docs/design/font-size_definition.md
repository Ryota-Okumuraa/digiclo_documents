# フォントサイズ定義（font-size_definition）

| 種別       | フォントサイズ | 行間     |
|------------|----------------|----------|
| 大見出し   | 32px           | auto     |
| 中見出し   | 24px           | auto     |
| 小見出し   | 16px           | auto     |
| 零見出し   | 12px           | auto     |
| 本文       | 16px           | 24px     |
